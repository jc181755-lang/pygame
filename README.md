# pygame-examples

This repository contains example scripts using [Pygame](https://www.pygame.org/), a popular Python library for building games and multimedia applications.

## Getting Started

1. Install Python (>=3.7)
2. Install Pygame:
   ```
   pip install pygame
   ```
3. Run an example:
   ```
   python example.py
   ```

## Contents

- `example.py`: A simple Pygame window with a colored background.
